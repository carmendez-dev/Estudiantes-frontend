# Sistema de Gestión de Estudiantes - Frontend

Frontend en Svelte para el sistema de gestión de estudiantes.

## Instalación

```bash
cd frontend
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se ejecutará en `http://localhost:5173`

## Build

```bash
npm run build
```

## Requisitos

- Node.js 16+
- API Backend corriendo en `http://localhost:8000`

## Características

- Listar todos los estudiantes en tabla
- Crear nuevo estudiante
- Editar estudiante existente
- Eliminar estudiante
- Búsqueda en tiempo real
- Diseño responsive
- Paleta de colores personalizada
- Animaciones suaves

## Paleta de Colores

- Azul Oscuro: `#0B2E50`
- Azul Celeste: `#27C5DA`
- Azul Pastel: `#7A95D9`
- Turquesa: `#3AC0B8`
- Rojo Coral: `#EF5C52`
